\label{index_neff-computation}%
\Hypertarget{index_neff-computation}%
\hypertarget{index_autotoc_md9}{}\doxysection{\texorpdfstring{NEFF Computation}{NEFF Computation}}\label{index_autotoc_md9}
NEFF computation determines the effective number of homologous sequences within an MSA. It takes into account sequence similarities and provides a measure of sequence diversity. ~\newline
 The code takes command-\/line flags for input and subsequently computes the score for an MSA file. ~\newline
 The detailed usage documentation can be found on the \doxysectlink{neff_computation}{NEFF Computation}{0} page.

\label{index_msa-file-conversion}%
\Hypertarget{index_msa-file-conversion}%
\hypertarget{index_autotoc_md10}{}\doxysection{\texorpdfstring{MSA File Conversion}{MSA File Conversion}}\label{index_autotoc_md10}
The MSA File Conversion tool provides a convenient way to convert Multiple \doxylink{struct_sequence}{Sequence} Alignment (MSA) files between various supported formats. ~\newline
 This versatile utility simplifies the process of format conversion, ensuring compatibility across different bioinformatics software and analysis pipelines. Simply specify the input and output files along with their formats, and the tool will handle the conversion for you. ~\newline
 The detailed usage documentation can be found on the \doxysectlink{converter}{MSA File Conversion}{0} page.
