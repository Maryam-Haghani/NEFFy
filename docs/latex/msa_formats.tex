\hypertarget{msa_formats_autotoc_md37}{}\doxysubsection{\texorpdfstring{A2M}{A2M}}\label{msa_formats_autotoc_md37}
Each sequence is represented by two lines\+:
\begin{DoxyItemize}
\item The first line starts with \texorpdfstring{$>$}{>} followed by the sequence identifier and some other remarks.
\item The second line contains the aligned residues; Alignments are shown with\+:
\begin{DoxyItemize}
\item Inserts as lower case characters,
\item Matches as upper case characters,
\item Deletions as \textquotesingle{} -\/ \textquotesingle{}, and
\item Gaps aligned to inserts as \textquotesingle{} . \textquotesingle{}
\end{DoxyItemize}
\end{DoxyItemize}

Sequences are separated by \textquotesingle{}\texorpdfstring{$>$}{>}\textquotesingle{}. The remaining lines before next \textquotesingle{}\texorpdfstring{$>$}{>}\textquotesingle{} contain the aligned sequence, which\+:
\begin{DoxyItemize}

\item \textquotesingle{} . \textquotesingle{} and \textquotesingle{} -\/ \textquotesingle{} are equivalent. Aligned =\texorpdfstring{$>$}{>} Sequences have the same length.
\end{DoxyItemize}

\hypertarget{msa_formats_autotoc_md43}{}\doxysubsection{\texorpdfstring{STO (Stockholm)}{STO (Stockholm)}}\label{msa_formats_autotoc_md43}
It consists of\+:
\begin{DoxyItemize}
\item A header line containing format and version information.
\item Mark-\/up lines that start with "{}\#=\+GF,"{} "{}\#=\+GC,"{} "{}\#=\+GS,"{} or "{}\#=\+GR."{}
\item Alignment lines featuring the sequence name and its corresponding aligned sequence. Within these lines\+:
\begin{DoxyItemize}
\item Inserts are represented as lowercase characters,
\item Matches are indicated by uppercase characters, and
\item Gaps are denoted by either \textquotesingle{} . \textquotesingle{} or \textquotesingle{} -\/ \textquotesingle{}.
\end{DoxyItemize}
\end{DoxyItemize}

Additionally, the "{}//"{} line indicates the end of the alignment. Sequences in this format are divided into segments of 200 characters.

\hypertarget{msa_formats_autotoc_md47}{}\doxysubsection{\texorpdfstring{CLUSTAL}{CLUSTAL}}\label{msa_formats_autotoc_md47}
Clustal is commonly associated with the Clustal series of programs for sequence alignment. The Clustal MSA format typically begins with a header line that provides information about the alignment.

Following the header, Clustal format represents each sequence as a pair of columns in a line.
\begin{DoxyItemize}
\item The first column contains the sequence name or identifier,
\item The second column contains the aligned sequence, which\+:
\begin{DoxyItemize}
\item Gaps are shown as \textquotesingle{}-\/\textquotesingle{}
\item Matches are shown as uppercase letters.
\end{DoxyItemize}
\end{DoxyItemize}

Sequences in this format are divided into segments of 60 characters.
