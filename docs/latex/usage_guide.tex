\item {\bfseries{Compute Asymmetric NEFF for RNA MSA\+:}} 
\begin{DoxyCode}{0}
\DoxyCodeLine{./neff\ -\/-\/file=example.a3m\ -\/-\/threshold=0.7\ -\/-\/norm=2\ -\/-\/is\_symmetric=false\ -\/-\/alphabet=1}

\end{DoxyCode}
 The resulting NEFF value will be displayed on the console based on the specified options.
\end{DoxyItemize}
\begin{DoxyItemize}
\item {\bfseries{Compute NEFF for Multimer MSA\+:}} 
\begin{DoxyCode}{0}
\DoxyCodeLine{./neff\ -\/-\/file=example.sto\ -\/-\/multimer\_MSA=true\ -\/-\/monomer\_length=20}

\end{DoxyCode}
 The tool will identify paired MSA sequences and sequences for the individual MSAs of monomers in the {\itshape example.\+sto} file which corresponds to a protein dimer, assuming the first monomer has a length of 20 residues. It will report NEFF values for the paired MSA as well as for the MSAs corresponding to unpaired sequences. If the provided MSA is not in the form of a multimer MSA, the tool will raise an error stating "{}\+MSA is not in the form of multimer MSA"{}.
\end{DoxyItemize}
\begin{DoxyItemize}
\item {\bfseries{Compute Column-\/wise NEFF\+:}} 
\begin{DoxyCode}{0}
\DoxyCodeLine{./neff\ -\/-\/file=example.aln\ -\/-\/column\_neff=true}

\end{DoxyCode}
 The result will display the NEFF value for each position in the alignment on a per-\/residue (column-\/wise) basis.
\end{DoxyItemize}
\begin{DoxyItemize}
\item {\bfseries{Random Masking and NEFF Calculation\+:}} 
\begin{DoxyCode}{0}
\DoxyCodeLine{./neff\ -\/-\/file=example.fasta\ -\/-\/mask\_enabled=true\ -\/-\/mask\_count=10\ -\/-\/mask\_percent=0.2}

\end{DoxyCode}
 The tool will mask 20\% of sequences in the example.\+fasta MSA file and repeat this process 10 times. The NEFF values from each iteration will be saved in the {\itshape neff\+\_\+values.\+txt} file. The masked MSA corresponding to the highest NEFF value will be saved in the {\itshape MSA\+\_\+with\+\_\+highest\+\_\+neff.\+fasta} file, representing the denoised MSA of the given example.\+fasta MSA, which can be used for downstream tasks requiring a high-\/quality MSA.
\end{DoxyItemize}
