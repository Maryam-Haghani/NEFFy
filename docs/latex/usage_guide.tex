\begin{DoxyItemize}
\item {\bfseries{Compute Asymmetric NEFF for an Integration of MSAs with Default Normalization (sqrt(\+L))\+:}} 
\begin{DoxyCode}{0}
\DoxyCodeLine{./neff\ -\/-\/file=../MSAs/uniref90\_hits.sto,../MSAs/bfd\_uniclust\_hits.a3m,../MSAs/mgnify\_hits.sto\ -\/-\/depth=2048\ -\/-\/is\_symmetric=false}

\end{DoxyCode}
 Result\+: \begin{quote}
MSA length\+: 338~\newline
 MSA depth\+: 2048~\newline
 NEFF\+: 106.\+067 \end{quote}

\end{DoxyItemize}

The tool will integrate MSAs from the list of files in the specified order and compute the NEFF value for the integrated MSA up to a depth of 2048. If the tool reaches the specified depth before all files are integrated, it will stop processing the remaining files.~\newline
 This process is akin to how Alpha\+Fold2.\+3 produces the final MSA for a protein by combining three distinct MSAs.~\newline
 ~\newline

The tool will detect a single repetition of the individual MSA of the chain within the given MSA file and report the NEFF value for it. If the provided MSA is not in the format of a multimer MSA of a homomer (i.\+e., the same MSA file repeated \textquotesingle{}n\textquotesingle{} times, where here n=2), the tool will raise an error. ~\newline

\begin{DoxyItemize}
\item {\bfseries{Compute NEFF for MSA of a Heteromer (L Normalization)\+:}} 
\begin{DoxyCode}{0}
\DoxyCodeLine{./neff\ -\/-\/file=../MSAs/heteromer.aln\ -\/-\/multimer\_MSA=true\ -\/-\/stoichiom=A2B1\ -\/-\/chain\_length=51,73\ -\/-\/norm=1}

\end{DoxyCode}
 Result\+: \begin{quote}
MSA sequence length\+: 175~\newline
 MSA depth\+: 3072~\newline
 NEFF of entire MSA\+:3.\+2836~\newline
 NEFF of Paired MSA (depth=1024)\+: 3.\+02721~\newline
 NEFF of Individual MSA for Chain A (depth=1024)\+: 4.\+37843~\newline
 NEFF of Individual MSA for Chain B (depth=1024)\+: 3.\+0194 \end{quote}

\end{DoxyItemize}

The tool will identify paired MSA sequences and the sequences for each individual MSA of chains in the given MSA file, assuming the first monomer has a length of 51 residues and the second has a length of 73. It will report NEFF values for the paired MSA as well as for the MSAs corresponding to unpaired sequences. If the provided MSA is not in the format of a multimer MSA of a heteromer, the tool will raise an error. ~\newline
