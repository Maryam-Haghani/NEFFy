To install the executable version of NEFFy\+:
\begin{DoxyEnumerate}
\item Clone the repository\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{git\ clone\ https://github.com/Maryam-\/Haghani/Neffy.git}

\end{DoxyCode}

\item Navigate to the project directory\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{cd\ Neffy}

\end{DoxyCode}

\item Compile the code using a C++ compiler that supports C++17 or a newer version, through the provided \href{https://github.com/Maryam-Haghani/Neffy/blob/main/Makefile}{\texttt{ Makefile}} in the repository. 
\begin{DoxyCode}{0}
\DoxyCodeLine{make}

\end{DoxyCode}

\begin{DoxyItemize}
\item {\bfseries{If the {\ttfamily make} command is not available on your operating system, refer to the \doxysectlink{help}{Help}{0} page.}}
\end{DoxyItemize}
\end{DoxyEnumerate}

Once the compilation is complete, you can run the program via the command line. ~\newline
 This package is cross-\/platform and works on Linux, Windows, and mac\+OS without requiring additional compilation.

To install the library from the source\+:


\begin{DoxyEnumerate}
\item Clone the repository\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{git\ clone\ https://github.com/Maryam-\/Haghani/Neffy.git}

\end{DoxyCode}

\item Navigate to the project directory\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{cd\ Neffy}

\end{DoxyCode}

\item Ensure you have {\ttfamily setuptools} and {\ttfamily wheel} installed\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{pip\ install\ setuptools\ wheel}

\end{DoxyCode}

\item Build the source distribution and wheel\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{python\ setup.py\ sdist\ bdist\_wheel}

\end{DoxyCode}

\item Install the package from the root directory of the project\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{pip\ install\ .}

\end{DoxyCode}

\begin{DoxyItemize}
\item Alternatively, you can install the package directly from the built wheel file (in the {\ttfamily dist} directory)\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{pip\ install\ dist/neffy-\/0.1-\/py3-\/none-\/any.whl}

\end{DoxyCode}

\end{DoxyItemize}
\end{DoxyEnumerate}
