\doxysubsection{Detailed Description}
MSA File Converter. 

This program converts the format of an input Multiple \doxylink{struct_sequence}{Sequence} Alignment (MSA) file to the format of an output MSA file.

Usage\+: ./converter --in\+\_\+file=$<$input\+\_\+file$>$ --out\+\_\+file=$<$output\+\_\+file$>$ \mbox{[}options\mbox{]}

Options\+: --in\+\_\+file=$<$input\+\_\+file$>$ Path to the input MSA file --out\+\_\+file=$<$output\+\_\+file$>$ Path to the output MSA file --alphabet=

Valid alphabet of MSA; alphabet option (0\+: Protein, 1\+: RNA, 2\+: DNA) (default\+: 0) --check\+\_\+validation=\texorpdfstring{$<$}{<}true/false\texorpdfstring{$>$}{>} Perform validation on sequences (default\+: true)

For detailed instructions, please refer to the documentation at \href{https://maryam-haghani.github.io/NEFFy}{\texttt{ https\+://maryam-\/haghani.\+github.\+io/\+NEFFy}}.
