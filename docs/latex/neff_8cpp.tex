\doxysubsection{Detailed Description}
Neff Computation. 

This program computes Number of EFFective sequences (NEFF) for a multiple sequence alignment (MSA) file. NEFF is a measure of effective sequence number that takes into account the redundancy and similarity of sequences in the MSA. It is commonly used in bioinformatics to assess the diversity of a set of sequences.

Usage\+: ./neff --file=$<$input\+\_\+file$>$ \mbox{[}options\mbox{]}

Options\+: --file=$<$input\+\_\+file$>$ Input files (comma-\/separated, no spaces) containing multiple sequence alignments (required)~\newline
"{}   -\/-\/alphabet=$<$value$>$                Valid alphabet of MSA; alphabet option (0\+: Protein, 1\+: RNA, 2\+: DNA) (default\+: 0)\textbackslash{}n"{} --check\+\_\+validation=\texorpdfstring{$<$}{<}true/false\texorpdfstring{$>$}{>} Perform validation on sequences (default\+: false)~\newline
"{}   -\/-\/threshold=$<$value$>$               Threshold value of considering two sequences similar (default\+: 0.\+8)\textbackslash{}n"{} --norm=

NEFF normalization option (0\+: sqrt(\+Length of alignment), 1\+: Length of alignment, 2\+: No normalization) (default\+: 0)~\newline
"{}   -\/-\/omit\+\_\+query\+\_\+gaps=$<$true/false$>$    Omit gap positions of query sequence from all sequences for NEFF computation (default\+: true)\textbackslash{}n"{} --is\+\_\+symmetric=\texorpdfstring{$<$}{<}true/false\texorpdfstring{$>$}{>} Consider gaps in similarity cutoff computation (asymmetric) or not (symmetric) (default\+: true)~\newline
"{}   -\/-\/non\+\_\+standard\+\_\+option=$<$value$>$     Handling non-\/standard letters in the given alphabet (0\+: As\+Standard, 1\+: Consider\+Gap\+In\+Cutoff, 2\+: Consider\+Gap)\textbackslash{}n"{} --depth=

Depth of MSA to be cosidered in computation (default\+: depth of given MSA)~\newline
"{}   -\/-\/gap\+\_\+cutoff=$<$value$>$              Cutoff value for removing gappy positions, when \#gaps in position $>$= gap\+\_\+cutoff (default=1 \+: does not remove anything)\textbackslash{}n"{} --pos\+\_\+start=

Start position of each sequence to be considered in neff (inclusive (default\+: 1))~\newline
"{}   -\/-\/pos\+\_\+end=$<$value$>$                 Last position of each sequence to be considered in neff (inclusive (default\+: length of MSA sequence))\textbackslash{}n"{} --only\+\_\+weights=\texorpdfstring{$<$}{<}true/false\texorpdfstring{$>$}{>} Return only sequence weights, as \# similar sequence, rather than the final NEFF (default\+: false)~\newline
"{}   -\/-\/multimer\+\_\+\+MSA=$<$true/false$>$       Compute NEFF for a multimer MSA (default\+: false)\textbackslash{}n"{} --stoichiom=

Multimer stoichiometry (default\+: empty) --chain\+\_\+length=
\begin{DoxyItemize}
\end{DoxyItemize}of the chains in heteromer multimer (default\+: 0)~\newline
"{} --residue\+\_\+neff=\texorpdfstring{$<$}{<}true/false\texorpdfstring{$>$}{>} Compute per-\/resiue (column-\/wise) NEFF (default\+: false)

For detailed instructions, please refer to the documentation at \href{https://maryam-haghani.github.io/NEFFy}{\texttt{ https\+://maryam-\/haghani.\+github.\+io/\+NEFFy}}.
